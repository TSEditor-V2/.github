# 🚛 ETS2 / ATS Save Editor

A powerful and modern save game editor for **Euro Truck Simulator 2 (ETS2)** and **American Truck Simulator (ATS)**.

Manage, modify, and optimize your trucking profiles with advanced save editing tools designed for safety, speed, and reliability.

> Compatible with the latest versions of ETS2 and ATS on Windows.

---

## 📖 Overview

**ETS2 / ATS Save Editor** is a lightweight desktop application that allows players to view, edit, and manage game save data without manually modifying save files.

Whether you want to adjust money, experience, owned vehicles, garages, drivers, or profile progression, the editor provides a structured interface that helps reduce save corruption risks while maintaining maximum control over your game data.

## 🚀 Getting Started

1. Download the latest release from the Releases page.
2. Launch `ETS2-ATS-Save-Editor.exe`.
3. Select your game:

   * Euro Truck Simulator 2
   * American Truck Simulator
4. Choose a profile and save file.
5. Modify the desired values.
6. Click **Save Changes**.
7. Launch the game and enjoy your updated profile.

---

## 🔒 Automatic Backup System

To help protect your progress, every modification automatically creates a backup before any changes are written.

### Backup Includes

* Original save file copy
* Timestamped restore point
* Quick rollback support

### Backup Location

```text
Documents/
├─ Euro Truck Simulator 2/
│  └─ save_backups/
└─ American Truck Simulator/
   └─ save_backups/
```

If necessary, simply replace the edited save with one of the generated backup files.

---

## ⚠️ Save Editing Warning

Editing save files always carries some level of risk.

Potential issues caused by incorrect modifications may include:

* Corrupted save data
* Missing trucks or trailers
* Garage ownership issues
* Economy synchronization problems
* Profile loading failures
* Game crashes

Always verify your changes and keep backup copies of important profiles.
